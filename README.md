# Tanglit  
*A literate‑programming toolkit that unifies docs, executes code, generates slides and exports code from a single source.*

## Installation (Ubuntu)

1. Go to the [Tanglit releases page](https://github.com/TP-FIUBA-133/tanglit/releases) and download the latest `.AppImage` file.

2. Make it executable:
   ```bash
   chmod +x Tanglit-x86_64.AppImage
   ```

3. Run it:
   ```bash
   ./Tanglit-x86_64.AppImage
   ```

Tanglit should open with its Markdown editor interface.  
No installation required — it’s fully self-contained.
